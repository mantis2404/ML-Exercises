We generate more data from the original given data, using image flipping/rotation, zoom in/out,etc..
- helpful where data collection is costly (medical fields)
- reduces overfitting
> if our sample mainly containes images with specific feature then the model learns that feature and fails to perform well on untrained features
> - for e.g. if our training set has only cats looking right then the model will learn that and will fail to identify cats looking to right
